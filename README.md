# My first AI bot using OpenAi and OpenRouter

### Prerequisites
* Python 3.10+ installed 
* The uv project/package manager ([installation docs](https://docs.astral.sh/uv/getting-started/installation/))
* Access to a Unix-like shell


### Step to How to run it
1. For HTML\
```git clone https://github.com/prashant679/first_ai_bot.git```\
OR\
for SSH\
```git clone git@github.com:prashant679/first_ai_bot.git```

2. ```cd first_ai_bot```

3. Run ```uv init```

4. Create a virtual environment\
```uv venv```

5. Activate the virtual environment\
```source .venv/bin/activate```

6. Run the file main.py using\
```uv run main.py```
